Restart network interface by command lines


  • Stop the network interface eth0:

ifdown eth0

  • Start the network interface eth0:

ifup eth0

  • Check the network

ifconfig

  • To see your various network interfaces, use the command:

/sbin/ifconfig –a

  • Set auto start the network interface after boot (https://access.redhat.com/documentation/en-US/Red_Hat_Enterprise_Linux/6/html/Deployment_Guide/s1-networkscripts-interfaces.html)

Edit /etc/sysconfig/network-scripts/ifcfg-eth0

The following is a sample ifcfg-eth0 file for a system using a fixed IP address:

DEVICE=eth0

BOOTPROTO=none

ONBOOT=yes

NETMASK=255.255.255.0

IPADDR=10.0.1.27

USERCTL=no
