Linux/unix OS ››

- latest version of linux os provide new way of encrypting for ssh, so in most case, we have to upgrade to use the latest version of ssh client tool like : putty, BvSshClient-Inst

- authentication with private key (used in silent tool like vagrant)


create pair of key by:

ssh-keygen -t rsa

or

ssh-keygen -t dsa


this will create 2 files: id_rsa and id_rsa.pub


change id_rsa.pub to  ~/.ssh/authorized_keys

copy id_rsa to use in client and change to C:\Users\Linh\.vagrant.d\insecure_private_key
