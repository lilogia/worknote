HPe EM/Systinet ›› Rgif device ››

  1. Make Systinet server trust a DataPower server’s certificate
  • Download *.cer file from browser or use ssltool to get *.crt file

# ssltool serverInfo --url https://16.153.233.13:5550 --certFile dp-16.153.233.13.crt

  • Import into $SYSTINET_HOME/conf/client.truststore

# keytool -import -v -trustcacerts -alias dp-16.153.233.13 -file dp-16.153.233.13.crt -keystore client.truststore

  • Restart server
