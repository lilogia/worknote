  1. Setting proxy in wget


  • Way1: edit wgetrc file

use_proxy=yes

http_proxy=web-proxy.cup.hp.com:8080


  • Way2: directly in command

wget  …  -e  use_proxy=yes  -e  http_proxy=web-proxy.cup.hp.com:8080  …


(More & quick: http://stackoverflow.com/questions/11211705/setting-proxy-in-wget )


  • Use a proxy on the command line

export http_proxy=”http://[username:password]@ web-proxy.cup.hp.com:8080”

sample:


export http_proxy="http://proxy.bbn.hp.com:8080"


(http://askubuntu.com/questions/47379/how-to-use-a-proxy-on-the-command-line )
