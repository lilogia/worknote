branch

Version control ›› Git ››

1. switch the origin branch develop: git checkout develop


2. pull to get the latest new branches: git pull

... enter username/password


3. switch to the specific branch postgres_3: git checkout postgres_3


4. view info of branch: git branch -v


5. delete branch: git branch -D postgres_2
