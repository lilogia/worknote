  • What to put in the log message, or how do I commit fix defect #xxxxx now…
    Always use Global ID QCCR1J[0-9]+ to reference your records
    svn ci –m “fixing defect #QCCR1J12345: description (reviewer’s HP email)”

  • IE10 is not officially supported, but several people reported it works, if the compatibility mode is turned on:
    http://windows.microsoft.com/en-us/internet-explorer/use-compatibility-view#ie=ie-10
    If that doesn’t work, you can still keep IE10 and use the ALM Explorer instead:
    http://update.external.hp.com/qualitycenter/qc110/others/almexplorer/
    I also use it, because it’s basically an IE without addressbar etc. and those are anyhow not helpful in an application like ALM, where the browser back button just takes you out of the application and it makes it obvious in TaskManager etc. which process is ALM.
